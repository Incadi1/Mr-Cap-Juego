MR CAP

Es un video juego para niños, basado 
en una basurita de color azul con blanco
(es el color de reciclaje de plastico),
el cual a medida que agarra las botellas
correctas va aumentando de velocidad, 
si no selecciona la debida se le van 
quitando vidas.


IMPORTANTE: 
Para poder correr el juego correctamente
en la configuración "Graphics quality" debe
estar en "Medium" si no las teclas 
A y D no funcionaras.
